# IaC Drift Reconciliation SaaS (Skeleton)

Basic project structure for a Python-based, AWS serverless drift reconciliation platform with Terraform-managed infrastructure.

## Scope of this scaffold

- No business logic yet
- No deployed resources yet
- Directory and placeholder files only

## Top-level layout

- `infra/terraform`: IaC modules and environment stacks
- `services`: Python Lambda/control-plane service boundaries
- `ui/minimal-dashboard`: minimal frontend shell for later implementation
- `tests`: unit, integration, load, and fixtures
- `docs`: architecture and planning docs
- `.github/workflows`: CI/CD pipeline placeholders
